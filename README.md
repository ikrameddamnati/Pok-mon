////Pokémon App React Native
Application mobile développée en utilisant React Native pour afficher les informations sur les Pokémons en utilisant l'API PokéAPI pour récupérer les informations.

////Comment Installer ?
Clonez : git clone https://github.com/your-username/pokemon-app-react-native.git
Install dependencies : npm install ou yarn
Launch the App : npm start ou yarn start
The app will be launched on your smartphone or computer
